# 【政务】政务文档脱敏助手

## 简介

政务文档脱敏助手（"正则捕手"）是一个专业的文档信息提取与脱敏工具，能够根据自然语言指令从上传的图片或 PDF 中精准提取或脱敏敏感信息。具备 OCR 识别、正则匹配及文档转换能力，可自动处理手机号、身份证号、统一社会信用代码等关键数据，生成符合公文排版习惯的 DOCX 文件并提供下载。

## 功能特性

### 1. 多格式文档解析
| 输入格式 | 处理方式 |
|---------|---------|
| **图片** | OCR 识别提取全文文本 |
| **PDF** | 文本提取与分析 |
| **纯文本** | 直接内容处理 |

### 2. 敏感信息智能识别
支持自动识别与处理的常见敏感数据类型：
- **手机号**：中间四位掩码（138****5678）
- **身份证号**：中间八位掩码（320102********0011）
- **统一社会信用代码**：保留前后部分，中间星号代替
- **邮箱地址**：提取列表或脱敏
- **自定义字段**：支持自然语言描述自定义脱敏规则

### 3. 自然语言指令驱动
用户无需掌握正则语法，直接用自然语言描述需求：
- "找出所有手机号和身份证号并脱敏"
- "提取所有邮箱地址并统计数量"
- "把统一社会信用代码隐藏掉"

系统自动将指令转化为正则表达式执行提取或脱敏。

### 4. 结构化输出与 DOCX 下载

```
📄 文档信息
文件名：【】
文档类型：【会议纪要/公示文件/合同/其他】

🎯 提取需求
提取目标：【手机号/身份证号/统一社会信用代码/邮箱/自定义】
处理方式：【提取列表/脱敏替换/统计计数】

📊 提取结果摘要
| 类别 | 原始内容示例 | 脱敏后 | 状态 |

📥 下载文件
[点击下载脱敏文档（DOCX）]
```

### 5. 公文规范排版
- 中文字体：宋体（文鼎 PL 简报宋）
- 英文字体：Times New Roman（Liberation Serif）
- 输出格式：DOCX

## 使用示例

**示例 1：图片身份证脱敏**

用户上传身份证图片并指示脱敏手机号和身份证号。

Agent 调用 `analyze_image` 进行 OCR 识别，自动将手机号中间四位和身份证号中间八位替换为星号，生成包含脱敏后内容的 DOCX 文件并返回下载链接。

**示例 2：合同邮箱提取**

用户上传合同 PDF 要求提取所有邮箱地址并统计数量。

Agent 调用 `analyze_text_file` 解析 PDF，使用正则匹配所有邮箱地址，统计总数，生成提取报告 DOCX。

**示例 3：公示文件信用代码脱敏**

用户上传公示文件要求隐藏统一社会信用代码。

Agent 解析文件内容，将信用代码保留前 8 位和后 4 位、中间用星号代替，生成符合公文排版（宋体）的脱敏文档。

## 工作流程

```
用户上传文档（图片/PDF/文本）
  ↓
文档解析：
  ├─ 图片 → analyze_image（OCR）
  └─ 文本/PDF → analyze_text_file
  ↓
理解用户自然语言指令
  ↓
正则匹配执行（提取/脱敏/统计）
  ↓
生成结构化 Markdown 报告
  ↓
markdown_to_document → DOCX + 下载链接
```

## 核心原则

- **先解析后处理**：先完成文档解析，再执行提取或脱敏
- **自然语言驱动**：用户无需掌握正则语法，用自然语言描述需求
- **精准脱敏**：敏感数据掩码处理后保留格式可读性
- **公文规范**：输出文档符合政务公文排版习惯
- **完整可下载**：处理结果以 DOCX 文件提供下载
