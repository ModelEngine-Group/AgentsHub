# 人才补贴助手

## 简介

人才补贴助手是一个专业的人才补贴顾问，能够根据用户的学历、年龄及意向城市搜索并提取最新的租房补贴政策。支持生成补贴对比表格和可视化图表，直观展示各城市的补贴差异与申请门槛，并提供个性化的申请建议与决策方案。

## 功能特性

### 1. 城市补贴政策查询
- 支持同时对比最多 2 个意向城市
- 基于学历（本科/硕士/博士）、年龄、毕业院校精准匹配
- 联网搜索最新政策（关键词模板：`{城市} {学历} 租房补贴 2026`）
- 抓取官方详情页面获取准确数据

### 2. 补贴数据提取
从搜索结果的详情页面中提取：
- 月补贴金额
- 补贴期限
- 学历/年龄要求
- 申请门槛与条件
- 原文链接

### 3. 可视化对比
| 图表类型 | 用途 |
|---------|------|
| **柱状图** | 对比各城市月补贴金额（必选） |
| **决策流程图** | 基于用户画像推荐最优城市（必选） |

### 4. 个性化决策建议
- 结合毕业院校排名（QS 前 100 等）推荐城市
- 针对特定区域政策（如余杭区、浦东新区、海淀区）给出建议
- 院校信息缺失时主动追问

## 技术架构

### 模型信息
- **主模型**: deepseek-v4-pro
- **业务逻辑模型**: deepseek-v4-pro

### 工具集
| 工具名称 | 功能说明 |
|---------|---------|
| `exa_search` | 联网搜索各城市的租房补贴政策（每城市 1 次） |
| `fetch_markdown` | 抓取政策详情页面内容，提取具体补贴条款 |
| `generate_mermaid_diagram` | 生成补贴对比柱状图和决策流程图（共 2 次） |

### 外部集成
| 服务 | 用途 |
|------|------|
| **Mermaid MCP** | 柱状图与流程图生成引擎 |
| **Fetch URL MCP** | 政策详情页面抓取 |

## 工作流程

```
用户输入（学历、年龄、意向城市）
  ↓
提取城市列表（最多2个）、学历、年龄、毕业院校
  ↓
对每个城市：
  exa_search → fetch_markdown → 提取补贴数据
  ↓
生成对比表格（城市、月补贴、期限、总额、门槛）
  ↓
generate_mermaid_diagram → 柱状图（月补贴对比）
  ↓
generate_mermaid_diagram → 决策流程图（基于用户画像推荐）
  ↓
输出个性化建议 + 免责声明
```

## 使用示例

**示例 1：双城市对比**

用户输入：
```
我是硕士，26岁，毕业于浙江大学，想了解上海和杭州的租房补贴。
```

Agent 分别搜索上海和杭州的硕士补贴政策，抓取详情页：
- **上海**：2000 元/月，期限 3 年
- **杭州**：2500 元/月，期限 3 年

生成柱状图对比月补贴金额，决策流程图因用户为浙大（QS 前 100）背景推荐杭州余杭区。

**示例 2：单城市查询**

用户输入：
```
帮我查一下北京对博士毕业生的补贴政策，我30岁。
```

Agent 搜索北京博士补贴政策，抓取详情页：
- **北京**：3000 元/月，期限 5 年

生成单城市柱状图和决策流程图（建议确认是否入职重点企业以申请海淀区额外支持）。

**示例 3：缺失院校信息**

用户输入：
```
对比一下广州和武汉的硕士补贴，我28岁，学校忘了填。
```

Agent 正常搜索两地数据：
- **广州**：1800 元/月，期限 2 年
- **武汉**：2000 元/月，期限 3 年

生成对比图后，因缺少院校信息，在回答末尾追问毕业院校以便提供更精准建议。

## 核心原则

- **真实数据**：基于联网搜索获取最新政策，禁止编造补贴数据
- **每城一搜**：每个城市仅搜索 1 次，总步骤控制在 8 步以内
- **图表必出**：每次回答必须包含柱状图和决策流程图
- **如实告知**：搜索无果时告知用户访问当地人社局官网核实
- **追问补全**：关键信息缺失时主动追问，不跳过
- **免责声明**：政策可能变动，以官方最新文件为准
