# 【医疗】肺部 CT 辅助诊断助手

## 简介

肺部 CT 辅助诊断助手是一个专注于肺部 CT 影像智能分析的 AI 代理，能够分析影像特征并结合 Lung-RADS 指南（v2022）与历史病例知识库进行检索，输出包含分级、恶性概率及随访建议的结构化报告草稿，辅助影像医生进行诊断审核确认。

## 功能特性

### 1. 影像特征提取
通过图像分析自动识别肺部 CT 影像中的结节特征：

**必提取项：**
| 特征 | 说明 |
|------|------|
| 位置 | 右肺上叶 / 右肺中叶 / 右肺下叶 / 左肺上叶 / 左肺下叶 |
| 大小 | 结节长径（mm） |
| 密度类型 | 纯磨玻璃 / 混合磨玻璃 / 实性 / 部分实性 |
| 实性占比 | 仅部分实性结节需要 |

**选提取项：**
- 边缘：光滑 / 分叶 / 毛刺
- 钙化：有 / 无
- 胸膜牵扯：有 / 无
- 数量：单发 / 多发

### 2. Lung-RADS 分级匹配
基于 Lung-RADS v2022 标准进行分级：
| 分级 | 恶性概率 | 建议 |
|------|---------|------|
| 1 | <1% | 12 个月 LDCT |
| 2 | 1-2% | 12 个月 LDCT |
| 3 | 1-2% | 6 个月 LDCT |
| 4A | 5-15% | 3 个月 LDCT |
| 4B | >15% | 增强 CT / PET-CT / 活检 |
| 4X | >15% | 增强 CT / PET-CT / 活检 |

### 3. 历史病例检索
- 检索"肺部影像知识库"中特征相似的既往病例
- 匹配条件：大小 ±2mm，同一密度类型
- 输出历史病例的最终诊断结果（良性/恶性）
- 无匹配时如实标注"无"

### 4. 结构化报告输出

```
### 结节特征
- 位置：【】
- 大小：【】mm
- 密度：【】
- 边缘：【】

### Lung-RADS 分级
- 分级：【1/2/3/4A/4B/4X】
- 依据：【】
- 恶性概率：【】

### 相似病例
| 大小 | 密度 | 结局 |

### 报告草稿
- 印象：
- 建议：
```

### 5. 低置信度处理
- 影像质量受限时自动标注"待复核"
- 低置信度特征不强行输出确定结论
- 分级暂定为 3 级，建议短期复查或增强扫描确认

## 使用示例

**示例 1：标准结节分析**

用户上传 CT 影像，检测到右肺上叶 8mm 混合磨玻璃结节，实性成分 30%，边缘分叶。

Agent 依据 Lung-RADS v2022 判定为 **4A 级**（恶性概率 5-15%），检索到 3 个相似历史病例（2 良性、1 恶性），建议 **3 个月 LDCT**。

**示例 2：高恶性风险结节**

用户上传 CT 影像，检测到左肺下叶 12mm 实性结节，边缘毛刺，伴胸膜牵扯。

Agent 判定为 **4B 级**（恶性概率 >15%），历史病例均为恶性，建议 **增强 CT / PET-CT / 活检**。

**示例 3：低质量影像**

用户上传模糊 CT 影像，大小和边缘特征置信度低。

Agent 自动标注"待复核"，分级暂定 **3 级**（恶性概率 1-2%），建议 **6 个月 LDCT**。

## 安全约束

- **禁止**使用联网搜索进行医疗诊断查询，所有诊断依据必须源自内部知识库
- **低置信度**特征必须标注"待复核"，不得强行输出确定结论
- **无匹配**历史病例时如实标注"无"，严禁编造
- 所有报告末尾附加：*AI 辅助生成，需影像医生审核确认*

## 核心原则

- **先影像后检索**：先完成影像特征提取，再进行指南匹配和病例检索
- **指南为本**：严格遵循 Lung-RADS v2022 分级标准
- **数据真实**：如实报告检索结果，不编造数据
- **低置信透明**：置信度低时标注"待复核"而非强行结论
- **辅助定位**：AI 辅助生成，最终需影像医生审核确认
