# 【医疗】体检报告解读助手

## 简介

体检报告解读助手是一个智能 AI 代理，能够通过 OCR 技术识别并解读医疗体检报告（血常规、CT 影像报告等），生成图文并茂的分析报告，并提供个性化的健康改善建议。

## 功能特性

### 1. OCR 识别与分析
- 支持图片格式体检报告识别（血常规、CT 影像报告等）
- 支持文本格式体检报告内容提取
- 自动识别报告中的异常指标

### 2. 智能解读与建议
- 针对异常指标提供详细的健康指导
- 支持联网搜索最新的医学健康指南
- 生成个性化的改善建议

### 3. 数据可视化
通过集成的医疗数据可视化专家，自动生成 2-3 张专业图表：

| 图表类型 | 用途 |
|---------|------|
| **雷达图** | 展示核心健康维度的全局视角 |
| **柱状图** | 展示超出/低于正常范围的指标项目 |
| **鱼骨图** | 分析异常指标的可能原因 |
| **液态图** | 展示关键指标的完成度/达标率 |
| **箱线图** | 与同龄人数据进行对比分析 |

### 4. 用户流程
1. 用户上传体检报告（PDF/图片）
2. 智能识别异常指标（如"低密度脂蛋白 3.8mmol/L"）
3. 查询健康知识库，获取改善建议
4. 生成图表（全局视角 + 局部深入分析）
5. 输出完整的分析报告

## 技术架构

### 模型信息
- **主模型**: deepseek-ai/DeepSeek-V4-Flash
- **业务逻辑模型**: Qwen/Qwen3.5-397B-A17B

### 工具集
| 工具名称 | 功能说明 |
|---------|---------|
| `analyze_image` | OCR 识别上传的体检报告图片，提取文本和异常指标 |
| `analyze_text_file` | 提取并分析上传的体检报告文本内容 |
| `exa_search` | 联网查询异常指标的最新健康指南和改善建议 |
| `report_data_vis_assistant` | 调用医疗数据可视化专家生成 2-3 张分析图表 |

### 依赖 Agent
- **report_data_vis_assistant** (ID: 2890): 医疗数据可视化专家，负责生成各类分析图表

## 使用示例

用户输入：
```
我上传了一份体检报告，帮我解读一下，并给出改善建议。
```

Agent 工作流程：
1. 调用 `analyze_image` 识别报告中的异常指标
2. 调用 `exa_search` 查询异常指标的改善方法
3. 调用 `report_data_vis_assistant` 生成雷达图和柱状图
4. 综合所有信息生成最终的图文分析报告
