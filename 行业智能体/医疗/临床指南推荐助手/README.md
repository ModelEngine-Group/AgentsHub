# 【医疗】《临床指南》推荐助手

## 简介

临床指南推荐助手（"指南引航员"）是一个专业的 AI 代理，能够基于患者的病情描述或诊断报告，快速、准确地推荐匹配的权威临床指南。具备信息提取、图像识别及知识库与联网双重检索能力，为临床决策提供精准支持。

## 功能特性

### 1. 智能信息提取
- **文本识别**：从患者病情描述中自动识别核心疾病/症状、学科领域及关键临床参数
- **图像 OCR**：支持上传化验单、诊断报告等图片，自动提取文本信息
- **参数提取**：年龄、检查指标、合并症、既往治疗等关键参数结构化输出

### 2. 多源指南检索
- **知识库优先检索**：优先匹配已入库的"临床指南库"权威指南
- **联网补充检索**：知识库结果不足时，联网获取最新发布/更新版本
- **时间限定**：优先近 5 年指南，必要时保留旧版本参考

### 3. 结构化推荐输出
按统一格式输出推荐结果：

| 模块 | 内容 |
|------|------|
| **信息提取结果** | 核心疾病、学科领域、关键参数 |
| **推荐指南** | 指南名称、发布机构、版本年份、核心推荐摘要、证据等级、获取链接 |
| **注意事项** | 版本冲突说明、更新提示、特殊人群注意 |
| **下一步建议** | 建议补充的精准推荐所需信息 |

### 4. 用户流程
1. 用户输入病情描述或上传诊断报告图片
2. Agent 提取核心疾病、关键参数（若为图片则先 OCR 识别）
3. 优先检索"临床指南库"知识库
4. 不足时联网补充检索最新指南
5. 筛选权威机构发布内容，按匹配度排序输出

## 使用示例

**示例 1：病情描述查询**

用户输入：
```
我父亲 65 岁，确诊 2 型糖尿病合并高血压，最近血糖控制不好，有什么指南推荐？
```

Agent 响应：
> **信息提取结果**
> - 核心疾病/症状：2 型糖尿病合并高血压
> - 学科领域：内分泌/心血管
> - 关键参数：年龄 65 岁、血糖控制不好、合并高血压
>
> **推荐指南**
> 1. 《糖尿病患者合并高血压管理专家共识（2023）》— 中国医师协会
> 2. 《中国 2 型糖尿病防治指南（2020 年版）》— 中华医学会糖尿病学分会

**示例 2：图片化验单查询**

用户上传化验单图片，Agent 调用 `analyze_image` 识别出"高脂血症（TC 6.5, LDL-C 4.2）"，然后检索知识库匹配《中国血脂管理指南（2023 年）》，输出含危险分层及他汀类药物使用建议的推荐结果。

**示例 3：罕见病查询**

用户查询法布雷病指南，知识库无匹配结果，Agent 自动联网搜索到《法布雷病早期筛查与诊断中国专家共识（2023）》并输出推荐，同时提示"本地知识库暂无收录，建议就医确认"。

## 核心原则

- **先提取后检索**：先明确患者病情参数，再进行指南匹配
- **知识库优先**：已入库的权威指南为首选来源
- **联网仅补充**：仅在知识库不足时进行联网检索
- **权威筛选**：仅保留权威机构发布的指南内容
- **结构化输出**：按统一格式呈现推荐结果，确保可读性
