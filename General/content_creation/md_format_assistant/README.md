# Markdown 排版助手

## 简介

Markdown 排版助手是一个内容格式化工具，能够根据用户提供的内容进行结构化整理与排版，将文本编排成清晰、美观、规范的 Markdown 格式，提升内容的可读性与呈现效果。

## 功能特性

### 1. 内容结构化排版
- 根据文本特点合理使用标题、列表、表格、引用、代码块等 Markdown 格式
- 支持在文字中适当添加 emoji 提升可读性
- 输出可直接复制的 Markdown 源码

### 2. 文件输出
- 支持通过 `create_file` 工具将排版后的内容保存为 `.md` 文件
- 文件路径使用相对路径，安全可控
- 默认使用 UTF-8 编码

### 3. 灵活的输出模式
| 场景 | 输出方式 |
|------|---------|
| 普通排版需求 | 直接在回复中输出可复制的 Markdown 源码 |
| 需要保存文件 | 调用 `create_file` 创建 `.md` 文件 |
| 内容不足 | 先向用户确认或生成合理模板，不盲目创建空文件 |

## 技术架构

### 模型信息
- **主模型**: glm-5.1
- **业务逻辑模型**: glm-5.1

### 工具集
| 工具名称 | 功能说明 |
|---------|---------|
| `create_file` | 创建文件并写入 Markdown 内容，支持自定义编码 |

## 工作流程

```
用户提供原始内容
  ↓
判断是否需要创建文件
  ├─ 仅排版 → 直接输出可复制的 Markdown 源码
  └─ 需要保存 → 调用 create_file 生成 .md 文件
  ↓
输出最终排版结果
```

## 使用示例

**示例 1：课程介绍排版**

用户输入：
```
把下面内容整理成漂亮的 Markdown 课程介绍：课程名 Python入门；对象 零基础；内容 变量、条件判断、循环、函数、文件读写；时长 4周；目标 能独立写小工具
```

Agent 将零散内容编排为结构清晰的 Markdown 课程介绍，包含课程简介、适合人群、课程内容、学习目标等模块，并适当加入 emoji。

**示例 2：会议纪要排版并保存文件**

用户输入：
```
请把以下会议纪要整理成 Markdown，并保存为 documents/meeting-summary.md：主题 Q2增长复盘；参会人 张三、李四、王五；结论 继续加大短视频投放；行动项 张三负责投放预算，下周五前完成
```

Agent 将会议纪要整理为 Markdown 格式，调用 `create_file` 保存到指定路径，同时在回复中提供可复制的 Markdown 源码。

**示例 3：项目计划排版**

用户输入：
```
帮我把这份项目计划改成 Markdown 格式：项目名称 客户反馈系统升级；背景 当前反馈分散在微信群和表格里；模块 反馈提交、标签分类、处理状态、数据统计；时间 9月1日启动，10月15日上线
```

Agent 将项目计划编排为结构清晰的项目文档，包含项目背景、目标、核心模块、时间安排等章节。

## 核心原则

- **源码优先**：默认在回复中直接输出可复制的 Markdown 源码
- **按需建文件**：只有用户明确要求保存文件时才使用 `create_file`
- **相对路径**：文件路径必须使用相对路径，禁止绝对路径
- **内容确认**：用户提供内容为空时先确认，不盲目创建空文件
- **规范编码**：文件编码默认使用 UTF-8
