# 【政务】政策解读助手

## 简介

政策解读助手（"政策透视员"）是一个专业的政策解读工具，能够将晦涩的政府文件转化为清晰易懂的白话解读。具备图像识别、文本分析及智能检索能力，可从上传的政策文件中提取关键信息，讲清政策背景、影响对象和行动指引。

## 功能特性

### 1. 多格式文件解析
| 输入格式 | 处理方式 |
|---------|---------|
| **图片** | OCR 识别提取政策文本 |
| **PDF** | 文本提取与分析 |
| **纯文本** | 直接内容处理 |

### 2. 政策关键信息提取
自动识别并提取政策文件的以下核心字段：
- 政策名称
- 发布机关
- 文号/日期
- 核心主题
- 关键条款

### 3. 多源关联检索
在提取政策基本信息后，自动执行关联检索：

| 检索源 | 目标内容 |
|-------|---------|
| **联网搜索** | 官方解读、答记者问、实施细则、上级法规 |
| **知识库** | 前序政策、配套办法、废止声明 |

> 注：当前版本未配置独立知识库，关联检索主要依赖联网搜索完成。

### 4. 结构化解读输出

```
🔍 政策基本信息
政策名称：【】
发布机关：【】
文号/日期：【】
核心主题：【】
关键条款：【】
```

## 技术架构

### 模型信息
- **主模型**: deepseek-ai/DeepSeek-V4-Flash
- **业务逻辑模型**: Qwen/Qwen3.5-397B-A17B

### 工具集
| 工具名称 | 功能说明 |
|---------|---------|
| `analyze_image` | OCR 识别政策文件图片中的文字内容 |
| `analyze_text_file` | 解析政策文件 PDF 或文本，提取标题、文号、核心条款等 |
| `exa_search` | 联网搜索官方解读、配套细则、历史关联政策 |

## 使用示例

**示例 1：图片政策文件解读**

用户上传政策文件图片，Agent 调用 `analyze_image` 提取标题"关于促进数字经济发展的指导意见"、发布机关"国家发改委"、文号"发改高技〔2023〕100 号"及核心条款，然后联网搜索官方解读和配套细则。

**示例 2：PDF 税收政策分析**

用户上传 PDF 税收优惠政策文件，Agent 提取"高新技术企业、所得税减免 15%、有效期至 2025 年底"等关键信息，联网搜索历史政策确认此前政策已于 2022 年底到期，新政策延续并扩大了适用范围。

**示例 3：无文件直接查询**

用户直接询问"最新的新能源汽车补贴政策"，Agent 跳过文件解析步骤，直接联网搜索获取 2024 年补贴标准，输出政策基本信息。

**示例 4：多文件对比分析**

用户上传两个环保标准文件，Agent 对比提取 PM2.5 标准差异（35 vs 30 微克/立方米），联网搜索标准收紧背景信息。

## 工作流程

```
用户输入（上传文件 / 直接提问）
  ↓
有文件？ → analyze_image / analyze_text_file → 提取政策基本信息
  ↓
无文件？ → 直接进入检索
  ↓
exa_search 联网搜索官方解读、配套细则、关联政策
  ↓
按标准格式输出解读报告
```

## 核心原则

- **先提取后检索**：先通过工具确认文件内容，再执行关联检索
- **精准提取**：明确提取政策标题、文号、发布机关、核心条款等关键字段
- **权威来源**：联网搜索仅限于官方解读、上级法规、地方配套细则等权威公开信息
- **事实准确**：所有事实性数据源自工具返回结果，严禁编造
- **白话转化**：将晦涩的政策语言转化为清晰易懂的解读
