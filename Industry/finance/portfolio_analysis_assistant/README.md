# 【金融】持仓分析助手

## 简介

持仓分析助手是一个投资组合诊断工具，能够根据用户的持仓数据生成资产配置与行业分布饼图，计算持仓集中度、最大回撤、年化收益等风险指标，并联网检索重大事件以提供风险预警，帮助用户全面了解持仓健康状况。

## 功能特性

### 1. 持仓数据接入
支持多种方式获取持仓信息：
- **图片识别**：上传基金/股票账户持仓截图，通过 OCR 提取持仓明细
- **手动输入**：用户直接输入持有的产品名称、份额、成本、市值
- **授权读取**：通过授权方式读取账户持仓数据

### 2. 持仓分析指标
| 指标 | 说明 |
|------|------|
| **持仓集中度** | 前 3 大仓位占总市值比例 |
| **资产配置比例** | 股票型/债券型/混合型/货币型占比 |
| **行业分布** | 各行业持仓占比，判断是否过度集中 |
| **最大回撤** | 持有期间的最大净值回撤幅度 |
| **年化收益** | 持有期间的年化收益率 |

### 3. 可视化图表
| 图表类型 | 用途 | 必选 |
|---------|------|------|
| **饼图** | 资产配置比例分布 | 是 |
| **饼图** | 行业分布比例 | 是 |
| **柱状图** | 前 N 大仓位占比对比 | 可选 |

占比类数据强制使用饼图展示，确保直观清晰。

### 4. 风险事件预警
通过联网搜索检索持仓产品的重大事件：
- 基金经理离职
- 产品清盘风险
- 负面舆情
- 市场重大波动

## 使用示例

**示例 1：资产配置分析**

用户输入：
```
我目前持有招商中证白酒指数基金5万元，易方达稳健收益债券基金3万元，天弘余额宝货币基金2万元，请帮我生成资产配置饼图。
```

Agent 分类为股票型 50%、债券型 30%、货币型 20%，生成资产配置饼图。

**示例 2：行业分布与集中度**

用户输入：
```
我的持仓包括：腾讯控股10万，贵州茅台15万，中国平安5万，招商银行10万。请帮我生成行业分布饼图，看看是否过度集中。
```

Agent 归类为互联网科技 25%、消费白酒 37.5%、金融 37.5%，生成行业分布饼图，提示消费白酒和金融行业集中度较高。

**示例 3：重仓事件预警**

用户输入：
```
我重仓了中欧医疗健康混合基金，帮我查查最近有没有什么重大事件，比如基金经理离职或者清盘风险？
```

Agent 联网搜索确认该基金无重大事件，基金经理仍为葛兰，运作正常。

**示例 4：综合分析**

用户输入：
```
我持有诺安成长混合10万，富国天惠成长混合5万，景顺长城新兴成长混合5万。请帮我分析资产配置，并查一下诺安成长混合最近有没有风险事件。
```

Agent 分析资产 100% 为混合型（配置较为激进），同时联网搜索发现诺安成长混合有基金经理离职传闻，提示短期波动风险。

## 工作流程

```
用户提供持仓信息（截图/手动输入/授权读取）
  ↓
提取持仓明细（名称、份额、成本、市值）
  ↓
计算分析指标：
  ├─ 持仓集中度（前3大占比）
  ├─ 资产配置比例
  ├─ 行业分布
  ├─ 最大回撤
  └─ 年化收益
  ↓
生成图表：
  ├─ generate_pie_chart → 资产配置饼图
  ├─ generate_pie_chart → 行业分布饼图
  └─ generate_column_chart → 前N大仓位对比（可选）
  ↓
exa_search 检索重大事件风险预警
  ↓
输出持仓分析报告 + 风险提示
```

## 核心原则

- **饼图优先**：占比类数据（资产配置、行业分布）必须使用饼图，禁止用柱状图或雷达图替代
- **数据准确**：图表数据严格基于实际持仓比例，确保准确无误
- **自身计算**：持仓集中度、最大回撤、年化收益等数值计算通过自身逻辑完成，不依赖图表或搜索工具
- **精准检索**：联网搜索必须包含具体产品名称及风险事件关键词，禁止搜索无关内容
- **风险预警**：对基金经理离职、清盘风险等重大事件进行联网检索并提示用户
