# 【金融】理财产品分析助手

## 简介

理财产品分析助手是一个智能投资分析工具，能够围绕基金或股票自动收集业绩、风险、持仓、基金经理、费用及舆情等多维度数据，并匹配同类产品池进行量化对比评分。结合风险控制指标给出明确的买入建议，并根据用户风险偏好输出仓位上限、持有期及买入方式建议。

## 功能特性

### 1. 多维度数据收集
自动覆盖六大维度数据：

| 维度 | 基金类数据源 | 股票类数据源 |
|------|-------------|-------------|
| **业绩** | 净值序列、日线行情 | 日线行情、财务指标 |
| **风险** | 最大回撤、波动率计算 | 最大回撤、波动率计算 |
| **持仓** | 基金持仓明细 | 前十大股东/流通股东 |
| **管理人** | 基金经理信息 | 管理层信息 |
| **费用** | 管理费、托管费、申购费等 | — |
| **舆情** | 新闻、研究报告 | 新闻、公告、调研记录 |

### 2. 同类产品池量化对比
- 基于基金类型/投资风格或行业/概念自动匹配可比池
- 同类样本不少于 5 只
- **五维度量化评分**：

| 维度 | 说明 |
|------|------|
| **收益** | 近 1/3/5 年收益率对比 |
| **风险** | 最大回撤、波动率与同类中位数比较 |
| **性价比** | 夏普比率、收益风险比 |
| **管理人** | 基金经理从业年限、历史业绩 |
| **费用** | 管理费、托管费、申购费水平 |

### 3. 风控研判与投资建议
基于综合评分和风险控制指标输出明确结论：

| 结论 | 条件 |
|------|------|
| **建议买入** | 综合评分高，最大回撤不超过同类中位数，数据完整 |
| **谨慎** | 回撤过大、持仓集中或数据部分缺失 |
| **不建议** | 风险显著高于同类、基本面恶化 |
| **数据不足无法判断** | 关键维度数据缺失 |

### 4. 个性化配置建议
当结论为"建议买入"时，结合用户风险偏好输出：
- **单支仓位上限**：保守型 ≤8%，平衡型 ≤20-30%，进取型 ≤40%
- **持有期建议**：基于波动率与历史表现
- **买入方式**：一次性建仓或分批定投

### 5. 可视化图表
基于真实数据生成：
| 图表类型 | 用途 |
|---------|------|
| 折线图/面积图 | 净值/行情趋势 |
| 雷达图 | 多维度对比评分 |
| 柱状图 | 同类排名 |
| 饼图 | 费用与持仓占比 |
| 词云图 | 舆情关键词 |
| 思维导图 | 流程结论 |

## 工作流程

```
用户提供标的（基金代码/股票代码）
  ↓
产品类型识别 → 选择对应数据接口
  ↓
多维度数据收集：
  ├─ 业绩（净值/行情）
  ├─ 风险（回撤/波动率计算）
  ├─ 持仓/重仓股
  ├─ 管理人/基金经理
  ├─ 费用
  └─ 舆情（近30天）
  ↓
同类产品池匹配（≥5只）
  ↓
五维度量化对比评分
  ↓
风控研判（最大回撤 vs 同类中位数）
  ↓
输出结论：建议买入 / 谨慎 / 不建议
  ↓
（若买入）结合风险偏好 → 仓位上限 + 持有期 + 买入方式
  ↓
风险提示
```

## 使用示例

**示例 1：基金分析**

用户输入：
```
帮我分析下基金 005827
```

Agent 识别为混合型基金（易方达蓝筹精选混合），获取净值序列、持仓（白酒/医药）、基金经理（张坤，从业 10 年+）、费用等数据，与同类混合型基金进行五维度评分。综合评分 78 分，但最大回撤 -42% 超过同类中位数 -30%，结论为"谨慎"，建议平衡型投资者单基仓位 ≤10%，持有期 ≥3 年，分批定投。

**示例 2：股票分析**

用户输入：
```
分析下贵州茅台 600519
```

Agent 通过股票分析助手进行多维度分析，获取近 3 年业绩、最大回撤 -38%（优于白酒行业中位数 -45%）、ROE 长期 30%+、PE 25 倍处于历史中位。综合评分 85 分，结论为"建议买入"，建议仓位上限 15%，持有期 ≥3 年，分批建仓。

**示例 3：ETF 分析**

用户输入：
```
帮我看下沪深300ETF 510300怎么样
```

Agent 获取 ETF 基础信息（华泰柏瑞沪深300ETF，规模超千亿，管理费 0.5%），与同类宽基 ETF 对比。综合评分 82 分，结论"建议买入"，建议仓位上限 20%，持有期 ≥2 年，推荐月定投。

**示例 4：指定风险偏好**

用户输入：
```
分析下中证500指数基金 000478，我是稳健型投资者
```

Agent 按稳健型偏好输出更保守的配置建议：单基仓位上限 8%，持有期 ≥3 年，强烈建议分批定投。

## 核心原则

- **标的明确**：仅当用户提供具体产品代码/名称后才调用数据工具
- **多维度全覆盖**：业绩、风险、持仓、管理人、费用、舆情缺一不可
- **真实数据**：所有评分数据必须来自工具返回，禁止编造数值
- **同类可比**：同类样本不少于 5 只，不足时需说明
- **风控为先**：最大回撤超过同类中位数时不得给出"建议买入"
- **历史不保证未来**：全程附带风险提示语
- **风险偏好确认**：未提供时默认平衡型并显式注明
- **数据缺失如实告知**：工具调用失败或字段缺失时如实说明，不用推测值填充
