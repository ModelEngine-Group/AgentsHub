# 行业雷达

## 简介

行业雷达是一个行业情报分析助手，能够全面追踪并整理目标行业的最新动态，涵盖巨头动向、投融资、政策变化及市场争议。它具备结构化搜索与信息溯源能力，确保每条信息都有具体主体、动作与数据支撑，为投资者提供高信息密度、多维度的深度周报。

## 功能特性

### 1. 全面行业动态追踪
覆盖以下维度：
- **巨头动向**：头部公司的战略、产品、人事等动作
- **投融资**：融资轮次、估值、投资方等信息
- **政策变化**：监管、立法、产业政策动态
- **市场争议**：行业内的争论、风险事件

### 2. 结构化搜索与信息溯源
- 每条新闻、每条动态都要有具体主体、动作与数据
- 链接以 Markdown 链接形式标注，方便溯源

### 3. 高覆盖的深度周报
- 为投资者提供高信息密度、多维度的深度周报
- 帮助把握行业趋势与潜在风险

## 工作流程

```
确定目标行业
  ↓
多轮联网搜索：巨头动向 / 投融资 / 政策 / 争议
  ↓
对每条信息做主体、动作、数据三要素校验
  ↓
按维度归类整理，补充链接溯源
  ↓
输出结构化深度周报
```

## 使用示例

**示例 1：单行业周报**

> 请帮我整理一下本周 AI 行业的关键动态

Agent 围绕 AI 行业进行多轮搜索，按巨头动向 / 投融资 / 政策 / 市场争议四个维度整理，每条信息附原文链接。

**示例 2：交叉行业对比**

> 请对比一下 AI 行业和半导体行业本周的关键变化

Agent 分别检索两个行业，按统一维度整理后输出对比性周报。

## 核心原则

- **高覆盖**：尽可能触达行业的巨头、资本、政策与争议面
- **三要素**：每条信息明确主体、动作、数据
- **可溯源**：所有引用附原文链接
- **多维度**：按统一维度分类，避免平铺直叙
- **风险视角**：不只罗列利好，也包括争议与负面信号
- **结论可执行**：为投资者提供趋势判断与风险提示
