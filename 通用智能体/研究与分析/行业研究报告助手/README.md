# 行业研究报告助手

## 简介

行业研究报告助手是一个专业的报告生成工具，具备信息搜索与深度分析能力，能够根据确定的研究范围，撰写行业概览、分析竞争格局、展开可比公司研究并筛选投资想法，帮助用户快速生成专业详尽的行业研究报告。

## 功能特性

### 1. 五步研究框架
| 步骤 | 内容 | 说明 |
|------|------|------|
| **1. 确定范围** | 行业/主题/角度/公司范围 | 明确研究边界，缺失信息时先行搜索 |
| **2. 行业概览** | 市场规模、发展趋势 | 梳理行业基本面与宏观环境 |
| **3. 竞争格局** | 主要参与者、市场份额 | 分析行业竞争态势与头部企业 |
| **4. 可比公司** | 估值对比、财务指标 | 展开可比公司深度分析 |
| **5. 投资想法** | 投资机会筛选 | 基于以上分析生成投资建议 |

### 2. 信息检索与整合
- 通过 `exa_search` 获取最新的行业动态、市场数据和竞品资讯
- 搜索结果经分析、提炼和整合后写入报告对应章节
- 禁止直接复制原始搜索结果作为报告最终内容

### 3. 结构化报告输出
```
一、行业概览
   - 市场规模与增长趋势
   - 技术发展与驱动因素
   - 政策与监管环境

二、竞争格局
   - 主要参与者及市场份额
   - 竞争优劣势分析
   - 新进入者与替代威胁

三、可比公司分析
   - 估值对比（PE/PB/PS）
   - 财务指标对比
   - 业务结构对比

四、投资想法
   - 产业链机会
   - 推荐标的及核心理由
   - 风险提示
```

### 4. 验证机制
| 参数 | 值 |
|------|-----|
| 验证启用 | 是 |
| 通过分数 | 0.75 |
| 严格度 | balanced（平衡） |
| 失败策略 | repair_then_controlled_summary（修复后受控总结） |

## 工作流程

```
用户提出行业研究需求
  ↓
步骤1：确定研究范围（行业/主题/角度/公司）
  ↓  (背景知识不足时先 exa_search)
步骤2：撰写行业概览（市场规模、发展趋势）
  ↓  (exa_search 获取数据后由技能生成)
步骤3：分析竞争格局（主要参与者、份额）
  ↓  (exa_search 获取竞品信息)
步骤4：展开可比公司研究（估值、财务对比）
  ↓  (exa_search 获取财务指标)
步骤5：筛选投资想法
  ↓
输出完整行业研究报告
```

## 使用示例

**示例 1：新能源汽车行业**

用户输入：
```
帮我生成一份新能源汽车行业的研究报告
```

Agent 搜索行业最新动态（2023 年全球销量持续增长，中国渗透率超 30%），依次生成行业概览、竞争格局（比亚迪国内龙头、特斯拉全球领先）、可比公司分析（比亚迪估值较低、特斯拉估值较高、蔚来亏损收窄）和投资想法（建议关注上游锂矿和电池厂商）。

**示例 2：AI 大模型行业**

用户输入：
```
我想了解人工智能大模型行业的投资机会，请帮我生成一份报告
```

Agent 搜索 AI 大模型行业现状（OpenAI、谷歌、百度纷纷推出大模型，算力需求激增），生成完整报告，投资想法建议关注算力芯片龙头及 AI 应用落地较快的公司。

**示例 3：创新药行业（GLP-1）**

用户输入：
```
请帮我做一份关于创新药行业的研究报告，重点关注GLP-1药物
```

Agent 搜索 GLP-1 药物研发进展（诺和诺德、礼来布局，减肥市场空间巨大），生成报告，投资想法建议关注 GLP-1 产业链相关 CDMO 企业及有同类创新药管线的公司。

## 核心原则

- **搜索先行**：每步内容生成前必须通过 `exa_search` 获取真实数据，禁止凭空编造
- **搜索词精准**：query 必须包含目标行业、公司名称或特定分析维度
- **整合输出**：搜索结果须经分析提炼后写入报告，禁止直接复制
- **唯一工具**：唯一外部工具为 `exa_search`，技能名对应内容生成指引而非可调用工具
- **范围明确**：研究范围缺失时先搜索确认，不凭空界定
