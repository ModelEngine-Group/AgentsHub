# 守法遵从官

## 简介

守法遵从官是一个法律智能检索助手，能够根据用户的问题解析并匹配相关法规，查找对应法条内容。它还可通过互联网搜索工具查询类似案例及其结果，最终以清晰美观的 Markdown 格式呈现"用户问题 / 相关法规 / 案例 / 结论"，让用户快速掌握法律要点。

## 功能特性

### 1. 法规匹配
- 解析用户提出的法律问题
- 依据大模型能力思考可能涉及的法律领域
- 用专业法律名词检索匹配具体法规

### 2. 法条原文回显
- 输出相关法规后，让大模型再次结合用户问题做判断
- 提供法规原文片段，方便用户核对

### 3. 类似案例检索
- 通过联网搜索工具查询类似案例
- 返回案例判决结果与参考意义

### 4. 结论综合
- 综合法规、案例与用户问题，给出最终结论
- 全部以 Markdown 标题分级呈现

### 5. 美观排版
- 内容分层清晰、视觉舒适
- 链接以 Markdown 链接形式给出，方便跳转

## 技术架构

### 工具集
| 工具名称 | 功能说明 |
|---------|---------|
| 法规检索工具 | 检索相关法规、法条原文 |
| 联网搜索工具 | 检索类似案例与判决结果 |

## 工作流程

```
用户提出法律问题
  ↓
解析问题，思考可能涉及的法律
  ↓
调用法规检索工具查找相关法规 / 法条
  ↓
大模型结合法规 + 用户问题，再次判断
  ↓
调用联网搜索工具查询类似案例
  ↓
按以下结构输出
  ├─ 一、用户问题
  ├─ 二、相关法规及原文
  ├─ 三、案例
  └─ 四、结论
```

## 使用示例

**示例 1：劳动争议**

> 试用期被辞退是否有补偿？

Agent 先解析可能涉及《劳动合同法》相关条款，检索法条原文，再搜类似案例，最终给出"问题 → 法规 → 案例 → 结论"的完整答复。

**示例 2：消费维权**

> 网购到假货怎么维权？

Agent 检索《消费者权益保护法》《电子商务法》等相关规定，搜类似维权案例，并输出清晰结论。

## 核心原则

- **法条优先**：法律问题先匹配法规原文，再辅以案例
- **二次判断**：法规检索后再次让模型结合用户问题做判断
- **案例支撑**：通过联网搜索补充真实类似案例
- **结构清晰**：以问题 / 法规 / 案例 / 结论四级标题呈现
- **美观排版**：Markdown 格式整洁，便于阅读
- **链接溯源**：所有引用附原文链接，方便跳转
