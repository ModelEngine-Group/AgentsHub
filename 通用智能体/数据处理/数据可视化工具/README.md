# 数据可视化工具

## 简介

数据可视化工具是一个数据可视化助手，能够根据用户提供的数据特性智能推荐并绘制柱状图、饼状图或散点图。它会分析数据维度与业务场景，判断最适合的图表类型，并调用相应工具生成可视化结果，让数据更直观、更易理解。

## 功能特性

### 1. 智能图表选型
根据数据特征自动选择最合适的图表类型：

| 图表类型 | 适用场景 |
|---------|---------|
| 柱状图 | 分类数据对比、时间序列趋势 |
| 饼状图 | 部分与整体的比例关系 |
| 散点图 | 两个连续变量之间的相关性或分布 |

### 2. 柱状图绘制（支持分组/堆叠）
- 接收 `category`、`value` 字段
- 支持通过 `group` 字段实现分组或堆叠展示
- 分组与堆叠不能同时启用

### 3. 饼状图绘制
- 接收 `category`、`value` 字段数组
- 自动计算各部分占比并绘制

### 4. 散点图绘制
- 接收 `x`、`y` 数值字段对象数组
- 在二维坐标系中绘制数据点

### 5. 输入校验
- 强制要求数据结构匹配工具规范
- 必要字段（如 `category`、`value`、`x`、`y`）缺失时报错

## 技术架构

### 模型信息
- **主模型**: deepseek-ai/DeepSeek-V3.2
- **业务逻辑模型**: deepseek-ai/DeepSeek-V3.2

### 工具集
需要先部署安装 mcp_chart 服务

| 工具名称 | 功能说明 |
|---------|---------|
| `generate_column_chart` | 生成柱状图（支持分组/堆叠） |
| `generate_pie_chart` | 生成饼状图 |
| `generate_scatter_chart` | 生成散点图 |

### 外部集成
| 服务 | 用途 |
|------|------|
| **chart MCP** | 图表绘制与渲染引擎 |

## 工作流程

```
用户提供数据
  ↓
分析数据结构（分类/比例/相关性）
  ↓
智能选择图表类型
  ├─ 比较类 → 柱状图
  ├─ 占比类 → 饼状图
  └─ 关系类 → 散点图
        ↓
按工具规范组装数据
  ↓
调用对应工具绘制
  ↓
返回图表 ID 给用户
```

## 使用示例

**示例 1：柱状图**

用户输入：
```
帮我用柱状图展示一下2023年各季度销售额：Q1 150万，Q2 200万，Q3 180万，Q4 220万
```

Agent 选用柱状图，调用 `generate_column_chart` 生成图表。

**示例 2：饼状图**

用户输入：
```
看看公司各部门的预算占比：研发35%、市场25%、销售20%、行政20%
```

Agent 选用饼状图，调用 `generate_pie_chart` 生成图表。

**示例 3：散点图**

用户输入：
```
帮我看一组身高和体重数据的关系
```

Agent 选用散点图，调用 `generate_scatter_chart` 生成图表。

**示例 4：分组柱状图**

用户输入：
```
对比北京和上海两地燃油车与电动车2023年的销量
```

Agent 选用分组柱状图，传入 `group=True`，对比两个分类维度。

## 核心原则

- **三图限定**：仅可使用柱状图、饼状图、散点图三种图表
- **按需选型**：根据数据特征主动判断最合适的图表类型
- **输入规范**：调用工具前确保数据结构与字段匹配工具规范
- **最小输出**：最终输出仅返回工具调用结果，不做额外解释
- **避免错用**：不调用列表外的工具，不擅自绘制其他图表类型
