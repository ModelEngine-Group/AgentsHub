# 提示词撰写师

## 简介

提示词撰写师是一个专业的意图澄清助手，专门为"跟 AI 说不清需求"的用户而设计。它能够通过结构化提问，由浅入深、由广到精地逐轮澄清用户模糊的意图，并把模糊描述转化为清晰、具体、可落地的执行要求，最终输出一份适合输入给大模型的精准提示词。

## 功能特性

### 1. 结构化多轮澄清
- 默认 10 轮，最多 20 轮（用户可自定义）
- 提问前先向用户确认轮数
- 每一轮仅提出 **1 个核心问题**，搭配 3-5 个预设答案

### 2. 由浅入深的层次感
- 先明确核心需求方向
- 再逐步细化各类细节要求
- 不跳步、不遗漏关键维度

### 3. 核心澄清维度（按需选取）
- 核心目标 / 用途
- 输出形式 / 载体
- 数量 / 规格要求
- 内容 / 风格要求
- 适用场景 / 受众
- 特殊限制 / 偏好

### 4. 多选题设计
- 每个问题覆盖常见场景
- 最后加"其他（请补充）"兜底
- 避免用户无方向作答

### 5. 最终输出
- 当所有关键维度都已确认后，输出一份清晰、可执行的 AI 提示词
- 既能直接喂给下游大模型使用，又能复盘哪些维度已经被澄清

## 工作流程

```
用户提出模糊需求
  ↓
向用户确认提问轮数（默认10轮，上限20轮）
  ↓
第 1 轮：明确核心目标 / 用途
  ↓
第 2~N 轮：按维度逐层深入
  ├─ 输出形式 / 载体
  ├─ 数量 / 规格要求
  ├─ 内容 / 风格要求
  ├─ 适用场景 / 受众
  ├─ 特殊限制 / 偏好
  ↓
关键维度均已澄清
  ↓
输出最终提示词（适合输入给大模型）
```

## 使用示例

**示例 1：营销方案澄清**

用户输入：
```
我想做一个营销方案。
```

Agent 先确认轮数（默认 10 轮），随后逐轮提问：核心目标、目标受众、预算范围、传播渠道、产出形式、时间节点、风格调性等。

**示例 2：产品介绍澄清**

用户输入：
```
帮我写一份产品介绍。
```

（假设用户回复"8 轮"）

- 第 1/8 轮：核心目标？A 吸引新客户购买 B 向现有客户介绍升级 C 内部培训 D 官网/宣传册内容 E 其他
- 第 2/8 轮：产品类型？A 实物 B SaaS 工具 C 课程 D 服务 E 其他
- …… 直至所有维度澄清后输出最终提示词

**示例 3：远程办公效率 PPT**

用户输入：
```
我需要一个关于"远程办公效率"的 PPT。
```

Agent 沿用 12 轮，第一轮聚焦主要用途（全员培训 / 高层汇报 / 行业见解 / 新员工引导 / 其他），随后逐轮澄清页数、风格、配色等。

**示例 4：Logo 设计澄清**

用户输入：
```
设计一个 logo。
```

Agent 通过 10 轮提问覆盖核心寓意、风格、使用场景、格式要求等。

**示例 5：线上活动策划澄清**

用户输入：
```
帮我策划一个线上活动。
```

Agent 进行 15 轮详细澄清，覆盖活动类型、目标受众、预算、平台、预期效果等。

## 核心原则

- **轮次上限**：默认 10 轮，最多 20 轮，先与用户确认
- **1 问 1 题**：每轮仅问 1 个核心问题，避免大段并列发问
- **多选兜底**：每题给出 3-5 个预设选项 + "其他（请补充）"
- **维度齐备**：核心澄清维度需覆盖目标/形式/规格/风格/受众/限制
- **无需工具**：整个流程由自然语言对话完成，不调用任何外部工具
- **最终交付**：澄清完成后输出一份适合直接输入给大模型的提示词
